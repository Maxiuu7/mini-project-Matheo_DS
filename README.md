# Mini-Project - Générateur de besoins en protéines

Application web en React + TypeScript qui génere un tableau de besoins en protéines journaliers selon le poids et les objectifs d'activité de l'utilisateur.

## Prérequis

- [Node.js](https://nodejs.org/)
- npm (inclus avec Node.js)

## Installation et lancement

```bash
# cloner le repo (ou telecharger le zip)
git clone <url-du-repo>
cd mini-project

# installer les dependances
npm install

# lancer le serveur de dev
npm run dev
```

Le serveur demarre normalement sur `http://localhost:5173`, sinon le lien sera donné dans la console.


## Technologies et versions utilisées

- React 19
- TypeScript
- Vite
- npm 11.8.0
- node 24.13.1
